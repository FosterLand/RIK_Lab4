# SurveyApp — Опитування "Розкажіть про себе"

Мобільний додаток-опитування, створений на **React Native** з використанням **Expo**.
Лабораторна робота №4.

## Опис

Додаток містить форму опитування про особисті вподобання із 7 запитань різних типів:

| #   | Запитання                              | Тип                        |
| --- | -------------------------------------- | -------------------------- |
| 1   | Скільки вам років?                     | Текстове поле              |
| 2   | Яке ваше улюблене хобі?                | Текстове поле              |
| 3   | Які види спорту вам подобаються?       | Чекбокси (множинний вибір) |
| 4   | Які жанри музики ви віддаєте перевагу? | Чекбокси (множинний вибір) |
| 5   | Які мови програмування ви знаєте?      | Мультивибір                |
| 6   | Які соціальні мережі ви використовуєте?| Мультивибір                |
| 7   | Що ви думаєте про майбутнє технологій? | Текстове поле              |

Після заповнення та натискання кнопки **"Надіслати"** відповіді зберігаються у файл `survey_results.txt`, який автоматично завантажується на диск.

Додаток використовує темну тему для комфортного використання.

## Технології

- **React Native** — фреймворк для мобільної розробки
- **Expo** — платформа для швидкого запуску та тестування
- **Expo Router** — файлова маршрутизація
- **JavaScript**

## Структура проєкту


## Запуск

### Передумови

- Node.js (v18+)
- npm

### Встановлення та запуск

```bash
# Перейти у папку проєкту
cd SurveyApp

# Встановити залежності (якщо ще не встановлені)
npm install

# Запустити у браузері
npx expo start --web

# Запустити через Expo Tunnel
npx expo start --tunnel
```

Додаток відкриється у браузері за адресою `http://localhost:8081`.

## Збереження результатів

Результати зберігаються у форматі txt.
